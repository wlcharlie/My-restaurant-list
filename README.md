## 開始使用

讓我們開始吧！

### 準備條件

使用專案需有
* 電腦
* 網路 *(若無Fontawesome的Icon會無法顯示)*
* Git Bash
* nvm & Node.js

### 安裝

1. 將專案複製到你的電腦上:
   ```sh
   git clone https://github.com/wlcharlie/My-restaurant-list.git
   ```
2. 進入到資料夾:
    ```sh
    cd My-restaurant-list
    ```
2. 用npm安裝套件 *這邊會安裝express/express.handlebars*
   ```sh
   npm install
   ```
3. 執行
   ```sh
   npm run dev
   ```
4. 確認出現這行提示代表成功：
    ```sh
    IS WORKING! Head to http://localhost:3000
    ```
    於瀏覽器輸入網址：http://localhost:3000
5. 結束
    * 在終端機上按下Crtl + C
